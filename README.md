# 🍜 Foodshop
Foods website system that can manage order, menu.

---

## 🍷 Features

### 🧑‍🍳 Admin & Chef
- Create, delete, edit table.
- Create, delete, edit order from each table.
- Create, delete, edit foods menu.
- Realtime order status check.

### 🧑‍🤝‍🧑 Customers
- Order foods to cart.
- Send orders to kitchen.
- Realtime order status check.

---

## 🧑‍💻 Tech Stack
- NextJS 15
- TailwindCss V.4
- PostgreSQL 17
- Prisma
